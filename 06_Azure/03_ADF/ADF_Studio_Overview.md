# ADF Studio Overview

## 1. What is ADF Studio

ADF Studio is the development environment of Azure Data Factory.
It is used to create pipelines, datasets, linked services, triggers and monitor executions.

## 2. Author Section

The Author section is used to create and manage pipelines, datasets and data flows.

## 3. Monitor Section

The Monitor section is used to track pipeline runs, activity status and errors.

## 4. Manage Section

The Manage section is used to configure linked services, integration runtimes and Git settings.

## 5. Publish Process

Changes are first saved as drafts and then published to make them available for execution.

## 6. Development Workflow

Manage → Dataset → Pipeline → Debug → Publish → Monitor

## 7. Enterprise Pipeline Lifecycle

Design → Develop → Debug → Publish → Schedule → Monitor
