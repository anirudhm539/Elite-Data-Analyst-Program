# ADF Security & Managed Identity

## Managed Identity
Azure-managed identity for secure authentication.

### Types
1. System Assigned
2. User Assigned

## RBAC
Role-Based Access Control.

## Common Roles

### Storage Blob Data Reader
Read only

### Storage Blob Data Contributor
Read and Write

### Storage Blob Data Owner
Full Storage Access

## Authentication vs Authorization

Authentication:
Who are you?

Authorization:
What can you do?

## Best Practices

- Use Managed Identity
- Follow Least Privilege
- Avoid Owner Role
- Review RBAC
