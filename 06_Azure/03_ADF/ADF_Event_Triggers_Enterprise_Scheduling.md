# ADF Event Triggers & Enterprise Scheduling

## What is Event Trigger
- Runs pipeline when an event occurs
- No fixed schedule required

## Blob Event Trigger
- Trigger fires when file arrives
- Common for vendor uploads

## Scheduling Types
1. Daily
2. Hourly
3. Event Based

## Enterprise Standards
- Do not run before data arrival
- Maintain dependency chains
- Use naming conventions
- Monitor all triggers

## Production Best Practices
- Test before publishing
- Enable retries
- Maintain logs
- Document triggers
