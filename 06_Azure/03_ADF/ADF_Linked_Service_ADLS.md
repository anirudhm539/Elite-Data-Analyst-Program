# ADF Linked Service - ADLS

## Purpose of Linked Service

A Linked Service is used to establish a connection between Azure Data Factory and external data stores.

## Linked Service Name

LS_ADLS

## Storage Account Connected

eliteanalyticsadls01

## Authentication Method

System-assigned Managed Identity

## Test Connection Result

Connection Successful

## Business Use Case

This Linked Service allows Azure Data Factory pipelines to read and write data from Azure Data Lake Storage Gen2.
