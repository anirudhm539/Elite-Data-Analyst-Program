# Azure Storage Account

## What is a Storage Account?

A Storage Account is a cloud-based service in Azure used to store and manage data securely and at scale.

## Why do companies use it?

- Centralized storage
- Secure access
- Backup and durability
- Scalable storage
- Integration with analytics tools

## Storage Services

### Blob Storage
Stores files and analytics data.

### File Storage
Provides shared folders in the cloud.

### Queue Storage
Stores messages between applications.

### Table Storage
Stores NoSQL structured data.

## Why Blob Storage is important for Data Analysts?

Blob Storage is commonly used to store CSV, Excel, JSON, and analytics datasets that are consumed by ETL pipelines, Power BI, and reporting systems.
