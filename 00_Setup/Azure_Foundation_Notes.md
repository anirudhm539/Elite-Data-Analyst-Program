# Azure Foundation Notes
## What is Cloud Computing?
Cloud Computing is the delivery of computing services such as servers, storage, databases, networking, software, and analytics over the internet instead of using your own physical computer or server.

It allows businesses to access resources on demand, pay only for what they use, and scale their infrastructure easily.

## What is Azure?
Microsoft Azure is a cloud computing platform created by Microsoft. It provides cloud services such as virtual machines, databases, storage, networking, analytics, artificial intelligence, and many other services.

Companies use Azure to build, deploy, manage, and scale applications without maintaining their own physical servers.

## IaaS vs PaaS vs SaaS
Cloud services are generally divided into three main categories:

- IaaS (Infrastructure as a Service)
- PaaS (Platform as a Service)
- SaaS (Software as a Service)
- ### IaaS (Infrastructure as a Service)

IaaS provides virtual infrastructure such as virtual machines, storage, and networking over the internet.

The cloud provider manages the physical hardware, while the customer manages the operating system, applications, and data.

**Example:** Azure Virtual Machines

### PaaS (Platform as a Service)

PaaS provides a complete platform for developing, testing, and deploying applications.

The cloud provider manages the infrastructure, operating system, and runtime, while the customer focuses only on the application and data.

**Example:** Azure App Service

### SaaS (Software as a Service)

SaaS provides ready-to-use software over the internet.

The cloud provider manages everything, and the customer simply uses the application.

**Example:** Microsoft 365

## What is a Resource?
A Resource is any individual service that you create and use in Azure.

Examples of Azure resources include:
- Virtual Machine
- Storage Account
- SQL Database
- Virtual Network
- App Service

Each resource performs a specific task and can be managed independently.

## What is a Resource Group?
A Resource Group is a logical container that holds related Azure resources.

It helps organize, manage, monitor, and control resources that belong to the same project or application.

Example:
An e-commerce application may have one Resource Group containing:
- Virtual Machine
- SQL Database
- Storage Account
- Virtual Network

- ## Why Companies Use Azure
- Companies use Azure because it provides:

- Scalability to handle growing workloads
- High availability and reliability
- Strong security features
- Cost-effective pay-as-you-go pricing
- Global data centers for faster performance
- Backup and disaster recovery solutions
- Easy integration with Microsoft products
