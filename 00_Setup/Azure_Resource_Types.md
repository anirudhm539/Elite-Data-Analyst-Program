# Azure Resource Types

## 1. Storage Account

### What it is
Azure Storage Account is a cloud storage service used to store files, blobs, backups, images, CSVs, and other data.

### Real-World Use Case
Store sales reports, customer files, Power BI source files, and data lake files.

### Cost Risk
Low

---

## 2. Azure SQL Database

### What it is
A managed cloud database service provided by Azure for storing structured data.

### Real-World Use Case
Store sales, customer, inventory, and transaction data for reporting and analytics.

### Cost Risk
Medium

---

## 3. Virtual Machine (VM)

### What it is
A cloud-based computer running Windows or Linux.

### Real-World Use Case
Run Python scripts, applications, automation jobs, or development environments.

### Cost Risk
High

---

## 4. App Service

### What it is
A platform for hosting web applications and APIs without managing servers.

### Real-World Use Case
Host internal company dashboards, portals, and business APIs.

### Cost Risk
Medium

---

## 5. Azure Functions

### What it is
A serverless service that runs code automatically when triggered.

### Real-World Use Case
Automate daily report generation, file processing, and scheduled tasks.

### Cost Risk
Low
